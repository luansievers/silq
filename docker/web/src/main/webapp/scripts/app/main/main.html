SILQ

Sistema de Integração Lattes - Qualis

SILQ, acrônimo para Sistema Integrado Lattes-Qualis, é uma ferramenta construída para auxiliar o processo de avaliação das produções científicas de pesquisadores de instituições de ensino superior.

A ferramenta utiliza a avaliação conceitual presente no Qualis para avaliar o peso das produções científicas desses pesquisadores.

SILQ oferece também recursos para que coordenadores de cursos de pós-graduação possam acompanhar o desempenho acadêmico dos pesquisadores associados. Controle de grupos e gráficos de evolução das produções científicas são alguns exemplos.
